"DigTheSounds" is a dashboard for displaying data returned from selected Spotify API user endpoints. Built using react, the dashboard allows users to view personal listening trends, user data stored by Spotify, and song metrics. My primary goal with this project was to integrate the OAuth2.0 process, Spotify API, and aspects of the MERN stack that I learned in Stanford's CS142 (Web Applications) course.

"EscapeStation" is virtual reality escape room experience developed for Stanford's VR art course. I was mainly responsible for scripting the events system, and developing the custom interactions that progress the experience. The experience was built using the Unity game engine for the Meta Quest 2, and incorporates industry standard development tools. Contact me for the Quest 2 APK build.

& Game Design

My artwork often meditates on the incompleteness of disparate family histories, and attempts to complete and elaborate on these histories. I create imagined spaces that reach across time and lineage, and appropriate the conventions of digital technology to tell stories. "The Trouble With Mt. Olive" is an interactive web game that tells one such story, and adopts the aesthetic of The Oregon Trail (1985) to do so. It was created using Twine, and standard web development tools.
